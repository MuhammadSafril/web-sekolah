# 🏫 Website Sekolah SMAN 1 Kendari

Website sekolah sederhana dibuat menggunakan **PHP + MySQL** dengan Laragon.
Dibuat untuk memenuhi tugas **Keamanan Data**.

## Fitur
- Halaman beranda dengan berita & statistik
- Data siswa & nilai
- Panel admin (login diperlukan)

## Cara Menjalankan
1. Install Laragon
2. Copy folder `sekolah` ke `C:\laragon\www\`
3. Import `database.sql` via phpMyAdmin
4. Buka `http://localhost/sekolah/`
